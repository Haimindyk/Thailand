# תאילנד 2027 🏝️ — אפליקציית מסלול (PWA)

אפליקציה להתקנה על הטלפון עם כל פרטי הטיול: טיסות, מלונות, מפה ותשלומים.
עובדת **לגמרי offline** — אחרי התקנה ראשונה אין צורך באינטרנט.

## מה יש בחבילה
- `index.html` — האפליקציה (כולל מפה מוטמעת, פונט מוטמע — אפס תלות חיצונית)
- `manifest.json` — הגדרות ההתקנה
- `sw.js` — Service Worker (אחראי על העבודה ללא אינטרנט)
- `icon-*.png` / `apple-touch-icon.png` — אייקונים

## איך להעלות ולהתקין

PWA חייב לרוץ דרך כתובת https (לא מקובץ מקומי). הדרך הכי פשוטה — Netlify (חינם, כמו ש-Bruto-Neto יושב עליו):

### העלאה ל-Netlify (הכי קל)
1. כנס ל-https://app.netlify.com/drop
2. גרור את **כל התיקייה** `thailand-pwa` לתוך החלון
3. תוך שניות תקבל כתובת כמו `https://random-name.netlify.app`
4. אפשר לשנות את השם בהגדרות האתר

### התקנה על הטלפון
- **אנדרואיד (Chrome):** פתח את הכתובת → יופיע כפתור "📲 התקן אפליקציה" למטה, או תפריט ⋮ → "התקן אפליקציה"
- **אייפון (Safari):** פתח את הכתובת → כפתור שיתוף → "הוסף למסך הבית"

אחרי ההתקנה האפליקציה תופיע עם אייקון על המסך הראשי ותיפתח במסך מלא, גם בלי אינטרנט.

## עריכת נתונים
בטאב "✏️ עריכה" אפשר להוסיף/לערוך/למחוק טיסות, מלונות ותשלומים.
לחיצה על "💾 שמור שינויים" שומרת מקומית בדפדפן (localStorage) — השינויים נשמרים בין פתיחות.

> שים לב: השמירה היא לכל מכשיר בנפרד. אם תרצה שהנתונים יסתנכרנו בין מכשירים — אפשר להוסיף בהמשך גיבוי לענן.
